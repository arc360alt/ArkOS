# ArkOS
A full version of ArkOS made with UBlue

## Features:
- Gnome Desktop
- Usefull tools build in
- Smooth Desktop Expereance
- Brew pre-installed
- Read-only FS so you cant mess up anything that much if you dont know what your doing
- Easy to customise and fork
- Probably buggy but thats fine

## Requierments:
- 15+GB of SSD or HDD Storage (SSD Reccomended)
- Electricity (optional)
- A CPU that was released before 2010
- A GPU That isnt an Nvidia one (This will be supported in later releases)
- A base level knowlage of Linux
